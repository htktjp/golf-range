# ゴルフ距離リング

航空写真の上に、50 / 100 / 150 / 200ヤードの距離リングを表示するWebアプリです。  
現在地を記録すると、自分の位置を線でつないでいきます  
線の保存・読込にも対応しています

※注意点　アプリは表示し続けないと、記録が取れません  
　　　　　表示を消さずにポケットに入れてご使用ください

## 主な機能

- 航空写真表示
- 距離リング表示
  - 50yd：赤
  - 100yd：青
  - 150yd：赤
  - 200yd：青
- 位置記録開始 / 終了
- 10秒ごとの位置記録
- 記録線のJSON保存 / 読込
- 読込後に全体表示へ自動調整
- 同じ場所に長くいた地点を青点表示

## 使い方

1. ページを開く  
2. **位置記録開始** を押す  
3. 移動すると線が記録される  
4. **位置記録終了** で停止  
5. **記録を保存** でJSON保存  
6. **記録を呼び出し** で再表示

## 補足

- 記録は10秒ごとに更新されます
- 終了を押し忘れても6時間で自動停止します
- GPS精度や航空写真のずれで数mの誤差が出ることがあります
